# LDKC Compiler

LDKC is a custom compiler for the LDK programming language.

##  Usage

### Compile a file
ldkc FILE_NAME.ldk -o FILE_NAME.exe

## Parameters

- `FILE_NAME.ldk` — source LDK file
- `-o FILE_NAME.exe` — output file name after compilation

## Example
ldkc main.ldk -o main.exe
## Note

Make sure `ldkc.exe` is either added to your system PATH or placed in the same directory as your project files.
